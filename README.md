# tih-showcase-projects
# Resolvet — TIH Showcase

## Description
Resolvet is a customer support and ticket management system designed to streamline client queries and improve resolution efficiency. Built as part of the TIH showcase, it demonstrates backend development, API integration, and deployment on Render.

## Features
- Ticket creation, assignment, and resolution tracking  
- CRM‑style client management  
- Secure login and user authentication  
- Deployed live on Render  

## Tech Stack
- Node.js, Express  
- JavaScript, CSS, HTML  
- Azure (cloud integration)  
- Render (deployment)  

## Live Demo
👉 [Resolvet on Render](https://resolvet.onrender.com)

# DialMetrics — TIH Showcase

## Description
DialMetrics is a performance dashboard designed to track outbound call metrics and visualize sales team efficiency. Built for the TIH showcase, it highlights frontend dashboard design and backend data handling.

## Features
- Real‑time call tracking and metrics visualization  
- CRM integration for lead management  
- Responsive dashboard UI  
- Deployed live on Render  

## Tech Stack
- Node.js, Express  
- React, JavaScript, CSS  
- Azure (cloud integration)  
- Render (deployment)  

## Live Demo
👉 [DialMetrics on Render](https://dialmetrics.onrender.com)
